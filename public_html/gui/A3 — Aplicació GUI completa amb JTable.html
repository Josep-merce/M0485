1. JTable i DefaultTableModel

Una JTable mostra dades en format de files i columnes, com un full de càlcul. S'usa quan volem mostrar informació estructurada: llistes d'alumnes, productes, usuaris, etc. El component de vista es separa del model de dades (DefaultTableModel), seguint el patró MVC.

Patró MVC a JTable: el DefaultTableModel és el model (les dades); la JTable és la vista (com es mostra); el nostre codi és el controlador (botó afegir, eliminar...). Qualsevol canvi al model es reflecteix automàticament a la vista.
1.1 Crear el model i assignar-lo a la taula
// Declaració a nivell de classe private DefaultTableModel modelTaula; // Al constructor, crear el model amb les capçaleres modelTaula = new DefaultTableModel( new String[]{ "Nom", "Categoria", "Preu", "Estoc" }, 0 ); taulaProductes.setModel(modelTaula); // Fer la taula no editable directament (opcional) modelTaula = new DefaultTableModel( new String[]{ "Nom", "Categoria", "Preu", "Estoc" }, 0 ) { public boolean isCellEditable(int row, int col) { return false; } };

2. Selecció de files i càrrega als camps

Un dels patrons més comuns amb JTable és: l'usuari selecciona una fila i els camps del formulari es carreguen automàticament per poder editar o eliminar. S'implementa amb un ListSelectionListener.

// Detectar quan l'usuari selecciona una fila → carregar als camps taulaProductes.getSelectionModel().addListSelectionListener(e -> { if (!e.getValueIsAdjusting()) { int fila = taulaProductes.getSelectedRow(); if (fila >= 0) { tfNom.setText((String) modelTaula.getValueAt(fila, 0)); tfCateg.setText((String) modelTaula.getValueAt(fila, 1)); tfPreu.setText( modelTaula.getValueAt(fila, 2).toString()); tfEstoc.setText( modelTaula.getValueAt(fila, 3).toString()); } } }); // Obtenir la fila seleccionada (retorna -1 si cap) int filaSelec = taulaProductes.getSelectedRow(); if (filaSelec < 0) { JOptionPane.showMessageDialog(this, "Selecciona una fila primer."); return; }

// AFEGIR una nova fila des del formulari private void btnAfegirActionPerformed(ActionEvent e) { String nom = tfNom.getText().trim(); String categ = tfCateg.getText().trim(); if (nom.isEmpty()) { JOptionPane.showMessageDialog(this,"Nom obligatori."); return; } double preu; int estoc; try { preu = Double.parseDouble(tfPreu.getText()); estoc = Integer.parseInt(tfEstoc.getText()); } catch (NumberFormatException ex) { JOptionPane.showMessageDialog(this, "Preu i estoc han de ser números."); return; } modelTaula.addRow(new Object[]{ nom, categ, preu, estoc }); netejarFormulari(); } // MODIFICAR la fila seleccionada private void btnModificarActionPerformed(ActionEvent e) { int fila = taulaProductes.getSelectedRow(); if (fila < 0) { JOptionPane.showMessageDialog(this,"Selecciona una fila."); return; } modelTaula.setValueAt(tfNom.getText(), fila, 0); modelTaula.setValueAt(tfCateg.getText(), fila, 1); modelTaula.setValueAt(Double.parseDouble(tfPreu.getText()), fila, 2); modelTaula.setValueAt(Integer.parseInt(tfEstoc.getText()), fila, 3); netejarFormulari(); } // ELIMINAR la fila seleccionada private void btnEliminarActionPerformed(ActionEvent e) { int fila = taulaProductes.getSelectedRow(); if (fila >= 0) modelTaula.removeRow(fila); }

E1 Gestor d'alumnes amb JTable Bàsic

Crea un formulari senzill de gestió d'alumnes amb JTable. Ha de permetre afegir alumnes i eliminar la fila seleccionada.

  1. Crea la taula amb les columnes: Nom, Cognoms, Nota. Model no editable directament.
  2. El formulari té tres JTextField (nom, cognoms, nota) i dos botons: Afegir i Eliminar.
  3. Afegir: valida que els camps no estiguin buits i que la nota sigui un número entre 0 i 10; afegeix la fila.
  4. Eliminar: comprova que hi hagi una fila seleccionada i la treu del model.
  5. En seleccionar una fila, els camps del formulari es carreguen automàticament.
Bàsic CA 5.6 · 5.7
Solució — Gestor d'alumnes amb JTable
// Fragment clau: validar i afegir alumne private void btnAfegirActionPerformed(ActionEvent e) { String nom = tfNom.getText().trim(); String cognoms = tfCognoms.getText().trim(); double nota; try { nota = Double.parseDouble(tfNota.getText()); } catch (NumberFormatException ex) { JOptionPane.showMessageDialog(this, "La nota ha de ser un número."); return; } if (nom.isEmpty() || cognoms.isEmpty()) { JOptionPane.showMessageDialog(this, "Tots els camps son obligatoris."); return; } if (nota < 0 || nota > 10) { JOptionPane.showMessageDialog(this, "Nota entre 0 i 10."); return; } modelTaula.addRow(new Object[]{ nom, cognoms, nota }); tfNom.setText(""); tfCognoms.setText(""); tfNota.setText(""); tfNom.requestFocus(); }

Solució — Aplicació de gestió completa amb persistència
// Guardar la taula en CSV private void guardarCSV(File fitxer) { try (PrintWriter pw = new PrintWriter(new FileWriter(fitxer))) { for (int i = 0; i < modelTaula.getRowCount(); i++) { pw.printf("%s;%s;%s;%s%n", modelTaula.getValueAt(i,0), modelTaula.getValueAt(i,1), modelTaula.getValueAt(i,2), modelTaula.getValueAt(i,3)); } JOptionPane.showMessageDialog(this, "Desat correctament."); } catch (IOException e) { JOptionPane.showMessageDialog(this, "Error: " + e.getMessage()); } } // Obrir CSV i carregar a la taula private void obrirCSV(File fitxer) { modelTaula.setRowCount(0); try (BufferedReader br = new BufferedReader(new FileReader(fitxer))) { String linia; while ((linia = br.readLine()) != null) { String[] p = linia.split(";"); if (p.length == 4) modelTaula.addRow(new Object[]{ p[0], p[1], p[2], Boolean.parseBoolean(p[3]) }); } } catch (IOException e) { JOptionPane.showMessageDialog(this, "Error: " + e.getMessage()); } }
