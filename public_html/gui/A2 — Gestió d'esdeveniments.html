1. El model d'esdeveniments de Swing

Swing usa el patró observador: un component és la font de l'esdeveniment, i un o diversos listeners en reben la notificació. Per afegir un listener a NetBeans, fem doble clic al component i el codi es genera automàticament. Manualment podem usar lambdes (Java 8+).

1.1 ActionListener — el més habitual

S'usa per botons, ítems de menú, combos i camps de text (Enter). El mètode a implementar és actionPerformed(ActionEvent e).

// Tres formes equivalents d'afegir ActionListener a un botó // 1. Classe anònima (estil clàssic) boto.addActionListener(new ActionListener() { public void actionPerformed(ActionEvent e) { System.out.println("Clicat!"); } }); // 2. Lambda (Java 8+, recomanada) boto.addActionListener(e -> System.out.println("Clicat!")); // 3. Referència a mètode (NetBeans genera this::nomMetode) boto.addActionListener(this::botoActionPerformed);
1.2 KeyListener — tecles
// Detectar tecla Enter en un JTextField tfNom.addKeyListener(new KeyAdapter() { public void keyPressed(KeyEvent e) { if (e.getKeyCode() == KeyEvent.VK_ENTER) { processarNom(); } } });

2. JList i DefaultListModel

Una JList mostra una llista d'ítems. Per afegir i eliminar elements dinàmicament cal usar un DefaultListModel<T> com a model de dades.

// Declaració del model a nivell de classe DefaultListModel<String> model = new DefaultListModel<>(); // Al constructor, assignar el model a la llista llistaTasques.setModel(model); model.addElement("Tasca inicial"); // Afegir un element private void botoAfegirActionPerformed(ActionEvent e) { String text = campTasca.getText().trim(); if (!text.isEmpty()) { model.addElement(text); campTasca.setText(""); } } // Eliminar l'element seleccionat private void botoEliminarActionPerformed(ActionEvent e) { int idx = llistaTasques.getSelectedIndex(); if (idx >= 0) model.remove(idx); } // Detectar quan canvia la selecció (ListSelectionListener) llistaTasques.addListSelectionListener(e -> { if (!e.getValueIsAdjusting()) { String sel = llistaTasques.getSelectedValue(); if (sel != null) etiquetaSeleccio.setText("Seleccionat: " + sel); } });

3. JTabbedPane i JMenuBar

3.1 JTabbedPane — pestanyes

Permet organitzar la interfície en pestanyes. A NetBeans s'afegeix a l'àrea de disseny i dins s'hi col·loquen els panels de cada pestanya. Per accedir programàticament a la pestanya activa:

// Obtenir l'índex de la pestanya seleccionada int idx = tabbedPane.getSelectedIndex(); // Navegar a una pestanya concreta tabbedPane.setSelectedIndex(1); // va a la 2a pestanya // Detectar canvi de pestanya tabbedPane.addChangeListener(e -> System.out.println("Pestanya: " + tabbedPane.getSelectedIndex()));
3.2 JMenuBar — menús
// A NetBeans: clic dret al JFrame → Add JMenuBar // Cada JMenu pot contenir JMenuItem, JCheckBoxMenuItem, JSeparator // Afegir mnemònic (Alt+A per al menú Arxiu) menuArxiu.setMnemonic(KeyEvent.VK_A); // Afegir accelerador (Ctrl+N per a "Nou") itemNou.setAccelerator( KeyStroke.getKeyStroke(KeyEvent.VK_N, InputEvent.CTRL_DOWN_MASK));

5. Exercicis

E1 Gestor de tasques amb JList Bàsic

Implementa l'aplicació de gestió de tasques de Canela (GestorTasquesLlistes.java). Permet afegir tasques amb prioritat, eliminar-les i filtrar entre pendents i fetes.

  1. Crea un formulari amb un JTextField per al text de la tasca i un JComboBox amb les prioritats: Alta, Mitjana, Baixa.
  2. Usa un DefaultListModel<String> com a model de la JList.
  3. El botó "Afegir tasca" afegeix l'element prioritat — text al model.
  4. El botó "Marcar com a fetes" mou les seleccionades a una llista de fetes a part.
  5. Els botons "Mostrar pendents" i "Mostrar fetes" filtren la JList.
Bàsic CA 5.7
Solució — Gestor de tasques amb JList
// Afegir tasca al model private void botoAfegirActionPerformed(ActionEvent evt) { String tasca = campTasca.getText().trim(); String prioritat = (String) comboPrioritat.getSelectedItem(); if (!tasca.isEmpty()) { model.addElement(prioritat + " — " + tasca); campTasca.setText(""); campTasca.requestFocus(); } } // Eliminar selecció private void botoEliminarActionPerformed(ActionEvent evt) { int[] selec = llistaTasques.getSelectedIndices(); for (int i = selec.length - 1; i >= 0; i--) model.remove(selec[i]); } // Detectar selecció i mostrar a l'àrea private void llistaTasquesValueChanged(ListSelectionEvent evt) { if (!evt.getValueIsAdjusting()) { String sel = llistaTasques.getSelectedValue(); if (sel != null) areaResultats.setText("Seleccionada: " + sel); } }

E3 App Login completa Avançat

Crea una aplicació de login amb validació, control de KeyPressed i navegació entre panells, seguint l'estructura dels exercicis 8-10 del curs UJB (Ubaldo Acosta, secció 17).

  1. Pantalla de login: JTextField usuari, JPasswordField contrasenya, botó "Entrar".
  2. Detecta la tecla Enter als dos camps amb KeyListener per activar el login sense fer clic.
  3. Valida: usuari = admin, contrasenya = 1234. Si falla, incrementa un comptador i bloqueja el botó al tercer intent.
  4. Si la validació correcta, oculta la finestra de login i mostra una finestra principal amb un menú i un missatge de benvinguda.
  5. Afegeix un botó "Tancar sessió" a la finestra principal que torna a mostrar el login.
Avançat CA 5.7 · 5.8
Solució — App Login completa
// Classe Login — validació i control d'intents public class LoginForm extends JFrame { private int intents = 0; private void intentarLogin() { String usuari = tfUsuari.getText().trim(); String contra = new String(pfContra.getPassword()); if (usuari.equals("admin") && contra.equals("1234")) { this.setVisible(false); new PantallaPrincipal(usuari, this).setVisible(true); } else { intents++; if (intents >= 3) { btnEntrar.setEnabled(false); JOptionPane.showMessageDialog(this, "Compte bloquejat. Massa intents fallits.", "Error", JOptionPane.ERROR_MESSAGE); } else { JOptionPane.showMessageDialog(this, "Credencials incorrectes. Intent " + intents + "/3", "Error", JOptionPane.ERROR_MESSAGE); pfContra.setText(""); pfContra.requestFocus(); } } } // KeyListener per detectar Enter als camps private void campKeyPressed(KeyEvent e) { if (e.getKeyCode() == KeyEvent.VK_ENTER) intentarLogin(); } }
