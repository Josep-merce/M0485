1. Fluxos de caràcters: visió general

Per treballar amb fitxers de text en Java s'utilitzen els fluxos de caràcters (character streams), que gestionen automàticament la conversió entre bytes i caràcters Unicode. Les classes base són Reader (lectura) i Writer (escriptura).

📖 Lectura (Reader)

  • FileReader — llegeix caràcters
  • BufferedReader — buffer + readLine()
  • Scanner — tokens i línies

✏️ Escriptura (Writer)

  • FileWriter — escriu caràcters
  • PrintWriter — escriu amb format
  • BufferedWriter — buffer eficient
Regla important: un fitxer sempre s'ha de tancar explícitament amb close() un cop acabada la lectura o escriptura. Si no es tanca, les dades escrites al buffer poden perdre's. La forma moderna és usar el bloc try-with-resources, que tanca automàticament.

2. Escriptura: FileWriter i PrintWriter

2.1 FileWriter — escriptura bàsica de caràcters

FileWriter és la classe més senzilla per escriure text en un fitxer. Si el fitxer no existeix, el crea; si existeix, el sobreescriu per defecte. Per afegir contingut al final (append), s'usa el constructor amb true.

import java.io.*; public class ExempleFileWriter { public static void main(String[] args) { // Escriptura normal (sobreescriu si existeix) try (FileWriter fw = new FileWriter("notes.txt")) { fw.write("Primera línia del fitxer\n"); fw.write("Segona línia del fitxer\n"); } catch (IOException e) { System.out.println("Error d'escriptura: " + e.getMessage()); } // Append: afegir contingut al final (segon paràmetre = true) try (FileWriter fw = new FileWriter("notes.txt", true)) { fw.write("Tercera línia afegida\n"); } catch (IOException e) { System.out.println("Error: " + e.getMessage()); } } }
2.2 PrintWriter — escriptura formatada

PrintWriter és molt més versàtil que FileWriter: ofereix println() per a tots els tipus primitius, printf() per formatar, i print(). S'acostuma a construir sobre un FileWriter.

import java.io.*; public class ExemplePrintWriter { public static void main(String[] args) { try (PrintWriter pw = new PrintWriter(new FileWriter("alumnes.txt"))) { pw.println("Anna Garcia"); // String + salt de línia pw.println(22); // int pw.println(8.75); // double pw.printf("%-15s %3d %.2f%n", "Bernat López", 19, 6.5); } catch (IOException e) { System.out.println("Error: " + e.getMessage()); } } }

3. Lectura: FileReader i Scanner

3.1 FileReader — lectura caràcter a caràcter

FileReader llegeix un caràcter a la vegada amb read(), que retorna un int (el valor Unicode del caràcter, o -1 si s'ha arribat al final). Sol ser poc eficient per si sol i normalment es combina amb un BufferedReader.

import java.io.*; try (FileReader fr = new FileReader("notes.txt")) { int c; while ((c = fr.read()) != -1) { System.out.print((char) c); // convertim int a char } } catch (IOException e) { System.out.println("Error: " + e.getMessage()); }
3.2 Scanner sobre fitxer — lectura per tokens i línies

Podem usar Scanner no només per llegir del teclat, sinó també per llegir d'un fitxer passant-li un objecte File o un FileReader. És molt pràctic per llegir fitxers amb dades estructurades.

import java.io.*; import java.util.Scanner; // Llegir línia a línia try (Scanner sc = new Scanner(new File("alumnes.txt"))) { while (sc.hasNextLine()) { String linia = sc.nextLine(); System.out.println(linia); } } catch (FileNotFoundException e) { System.out.println("Fitxer no trobat: " + e.getMessage()); } // Llegir dades estructurades (un enter i un double per línia) try (Scanner sc = new Scanner(new File("notes.txt"))) { while (sc.hasNextInt()) { int id = sc.nextInt(); double nota = sc.nextDouble(); System.out.printf("ID: %d → Nota: %.2f%n", id, nota); } } catch (FileNotFoundException e) { System.out.println("Error: " + e.getMessage()); }

4. Buffers: BufferedReader i BufferedWriter

Les classes BufferedReader i BufferedWriter afegeixen una memòria intermèdia (buffer) a les operacions d'E/S. En lloc de fer una operació de disc per cada caràcter, acumulen les dades en memòria i fan accesos al disc en blocs grans, cosa que millora molt el rendiment.

Per què usar buffers? Accedir al disc és molt lent. Amb un buffer, en lloc de fer 10.000 lectures d'1 byte cadascuna, es fa 1 lectura de 10.000 bytes. El rendiment pot millorar 10-100×.
4.1 BufferedWriter — escriptura eficient
import java.io.*; public class ExempleBufferedWriter { public static void main(String[] args) { try (BufferedWriter bw = new BufferedWriter(new FileWriter("sortida.txt"))) { for (int i = 1; i <= 5; i++) { bw.write("Línia número " + i); bw.newLine(); // salt de línia portable (\r\n o \n) } // El close() fa flush automàticament: buida el buffer al disc } catch (IOException e) { System.out.println("Error: " + e.getMessage()); } } }
4.2 BufferedReader — lectura eficient línia a línia
import java.io.*; public class ExempleBufferedReader { public static void main(String[] args) { try (BufferedReader br = new BufferedReader(new FileReader("sortida.txt"))) { String linia; int numLinia = 1; // readLine() retorna null quan s'acaba el fitxer while ((linia = br.readLine()) != null) { System.out.printf("%3d: %s%n", numLinia++, linia); } } catch (IOException e) { System.out.println("Error: " + e.getMessage()); } } }
4.3 Resum: quina classe usar?
Cas d'úsEscriptura recomanadaLectura recomanada
Fitxer de text senzillFileWriterFileReader + cast a char
Escriure tipus diversos (int, double...)PrintWriter—
Fitxer gran / rendimentBufferedWriterBufferedReader
Llegir dades estructurades (tokens)—Scanner(File)
Millor pràctica generalPrintWriter(new FileWriter(...))BufferedReader(new FileReader(...))

5. Exercicis

E1 Diari personal Bàsic

Crea un programa que permeti afegir entrades a un fitxer de diari. Cada vegada que s'executa, demana el text d'una nova entrada i l'afegeix al final del fitxer diari.txt amb la data del sistema.

  1. Llegeix el text de l'entrada amb Scanner.
  2. Obre el fitxer en mode append amb FileWriter(nom, true).
  3. Escriu la data (usa LocalDate.now()) i el text separats per " | ".
  4. Mostra el contingut actual del fitxer llegint-lo amb BufferedReader.
Bàsic CA 5.4 · 5.5
✅ Solució — Diari personal
import java.io.*; import java.time.LocalDate; import java.util.Scanner; public class DiariPersonal { static final String FITXER = "diari.txt"; public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Escriu la teva entrada d'avui: "); String entrada = sc.nextLine(); sc.close(); // Afegim al fitxer (append = true) try (PrintWriter pw = new PrintWriter(new FileWriter(FITXER, true))) { pw.printf("%s | %s%n", LocalDate.now(), entrada); } catch (IOException e) { System.out.println("Error en escriure: " + e.getMessage()); return; } // Mostrem el contingut actual System.out.println("\n=== Diari complet ==="); try (BufferedReader br = new BufferedReader(new FileReader(FITXER))) { String linia; while ((linia = br.readLine()) != null) { System.out.println(linia); } } catch (IOException e) { System.out.println("Error en llegir: " + e.getMessage()); } } }
E2 Comptador de paraules Mitjà

Crea un mètode que rebi el nom d'un fitxer de text i retorni el nombre de línies, paraules i caràcters que conté. Mostra els resultats per pantalla.

  1. Llegeix el fitxer línia a línia amb BufferedReader.
  2. Per cada línia, compta les paraules usant split("\\s+").
  3. Compta els caràcters sumant la longitud de cada línia.
  4. Mostra el resum final en format de taula.
Mitjà CA 5.4 · 5.5
✅ Solució — Comptador de paraules
import java.io.*; public class ComptadorParaules { public static void main(String[] args) { analitzarFitxer("diari.txt"); } public static void analitzarFitxer(String nomFitxer) { int linies = 0, paraules = 0, caracters = 0; try (BufferedReader br = new BufferedReader(new FileReader(nomFitxer))) { String linia; while ((linia = br.readLine()) != null) { linies++; caracters += linia.length(); if (!linia.trim().isEmpty()) { paraules += linia.trim().split("\\s+").length; } } } catch (IOException e) { System.out.println("Error: " + e.getMessage()); return; } System.out.println("\n=== Estadístiques de '" + nomFitxer + "' ==="); System.out.printf("%-12s %6d%n", "Línies:", linies); System.out.printf("%-12s %6d%n", "Paraules:", paraules); System.out.printf("%-12s %6d%n", "Caràcters:", caracters); } }
E3 Cerca i reemplaça Mitjà

Crea un programa que llegeixi un fitxer de text, reemplaci totes les ocurrències d'una paraula per una altra, i escrigui el resultat en un fitxer nou amb el sufix _modificat.txt.

  1. Demana per consola: nom del fitxer, paraula a cercar i paraula de reemplaç.
  2. Llegeix el fitxer original línia a línia amb BufferedReader.
  3. Per cada línia, aplica linia.replace(cerca, reemplaç).
  4. Escriu les línies modificades al fitxer nou amb PrintWriter.
  5. Mostra quantes substitucions s'han fet en total.
Mitjà CA 5.4 · 5.5
✅ Solució — Cerca i reemplaça
import java.io.*; import java.util.Scanner; public class CercaReemplaca { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Nom del fitxer: "); String nomFitxer = sc.nextLine(); System.out.print("Paraula a cercar: "); String cerca = sc.nextLine(); System.out.print("Paraula de reemplaç: "); String reemplac = sc.nextLine(); sc.close(); // Nom del fitxer de sortida int punt = nomFitxer.lastIndexOf('.'); String nomSortida = (punt >= 0) ? nomFitxer.substring(0, punt) + "_modificat" + nomFitxer.substring(punt) : nomFitxer + "_modificat"; int totalSubstitucions = 0; try (BufferedReader br = new BufferedReader(new FileReader(nomFitxer)); PrintWriter pw = new PrintWriter(new FileWriter(nomSortida))) { String linia; while ((linia = br.readLine()) != null) { // Comptem ocurrències en aquesta línia int count = linia.split(java.util.regex.Pattern.quote(cerca), -1).length - 1; totalSubstitucions += count; pw.println(linia.replace(cerca, reemplac)); } } catch (IOException e) { System.out.println("Error: " + e.getMessage()); return; } System.out.printf("✅ Fet. %d substitució(ns). Fitxer guardat: %s%n", totalSubstitucions, nomSortida); } }
E4 Gestió de notes d'alumnes Avançat

Crea un programa que gestioni un fitxer notes.csv on cada línia té el format nom;nota. El programa ha de permetre afegir alumnes, mostrar tots els alumnes i calcular la mitjana de notes.

  1. Mostra un menú: 1) Afegir alumne, 2) Mostrar tots, 3) Mitjana, 4) Sortir.
  2. L'opció 1 demana nom i nota i els afegeix al final del fitxer (append).
  3. L'opció 2 llegeix el fitxer i mostra els alumnes amb printf alineat.
  4. L'opció 3 llegeix totes les notes, les suma i calcula la mitjana.
  5. Valida que la nota estigui entre 0 i 10.
Avançat CA 5.3 · 5.4 · 5.5
✅ Solució — Gestió de notes d'alumnes
import java.io.*; import java.util.Scanner; public class GestioNotes { static final String FITXER = "notes.csv"; public static void main(String[] args) { Scanner sc = new Scanner(System.in); int opcio; do { System.out.println("\n=== Gestió de Notes ==="); System.out.println("1. Afegir alumne"); System.out.println("2. Mostrar tots"); System.out.println("3. Calcular mitjana"); System.out.println("4. Sortir"); System.out.print("Opció: "); opcio = sc.nextInt(); sc.nextLine(); switch (opcio) { case 1 -> afegirAlumne(sc); case 2 -> mostrarTots(); case 3 -> calcularMitjana(); case 4 -> System.out.println("Fins aviat!"); default -> System.out.println("Opció no vàlida."); } } while (opcio != 4); sc.close(); } static void afegirAlumne(Scanner sc) { System.out.print("Nom: "); String nom = sc.nextLine(); double nota; do { System.out.print("Nota (0-10): "); nota = sc.nextDouble(); sc.nextLine(); } while (nota < 0 || nota > 10); try (PrintWriter pw = new PrintWriter(new FileWriter(FITXER, true))) { pw.printf("%s;%.2f%n", nom, nota); System.out.println("✅ Alumne afegit."); } catch (IOException e) { System.out.println("Error: " + e.getMessage()); } } static void mostrarTots() { System.out.printf("%n%-20s %6s%n", "Nom", "Nota"); System.out.println("-".repeat(28)); try (BufferedReader br = new BufferedReader(new FileReader(FITXER))) { String linia; while ((linia = br.readLine()) != null) { String[] parts = linia.split(";"); if (parts.length == 2) System.out.printf("%-20s %6s%n", parts[0], parts[1]); } } catch (IOException e) { System.out.println("Fitxer buit o no trobat."); } } static void calcularMitjana() { double suma = 0; int count = 0; try (BufferedReader br = new BufferedReader(new FileReader(FITXER))) { String linia; while ((linia = br.readLine()) != null) { String[] parts = linia.split(";"); if (parts.length == 2) { suma += Double.parseDouble(parts[1].replace(",", ".")); count++; } } } catch (IOException e) { System.out.println("Fitxer buit o no trobat."); return; } if (count > 0) System.out.printf("Mitjana de %d alumnes: %.2f%n", count, suma / count); else System.out.println("No hi ha dades."); } }
