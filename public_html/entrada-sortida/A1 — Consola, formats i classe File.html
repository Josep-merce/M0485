1. Entrada per teclat amb Scanner

La classe Scanner del paquet java.util permet llegir dades de l'entrada estàndard (teclat). Es crea passant-li System.in com a font de dades i s'utilitzen mètodes específics per a cada tipus.

Recordatori: Scanner ja l'hem usat en activitats anteriors per llegir dades simples. Aquí aprofundim en tots els mètodes disponibles i en els problemes habituals com el salt de línia perdut.
1.1 Mètodes principals de Scanner
MètodeLlegeixExemple
nextInt()Un enterint edat = sc.nextInt();
nextDouble()Un decimaldouble nota = sc.nextDouble();
nextLong()Un enter llarglong dni = sc.nextLong();
next()Una paraula (fins espai)String mot = sc.next();
nextLine()Una línia completaString frase = sc.nextLine();
nextBoolean()Un booleà (true/false)boolean ok = sc.nextBoolean();
hasNextInt()Comprova si el següent és enterwhile (sc.hasNextInt())
1.2 Exemple complet: formulari de dades
import java.util.Scanner; public class FormulariDades { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Introdueix el teu nom: "); String nom = sc.nextLine(); System.out.print("Introdueix la teva edat: "); int edat = sc.nextInt(); System.out.print("Introdueix la teva nota (0.0 - 10.0): "); double nota = sc.nextDouble(); sc.close(); System.out.println("--- Resum ---"); System.out.println("Nom: " + nom); System.out.println("Edat: " + edat); System.out.println("Nota: " + nota); } }
Problema del salt de línia perdut: quan uses nextInt() o nextDouble() seguit de nextLine(), el primer nextLine() llegeix el \n que queda al buffer i retorna una cadena buida. Solució: afegeix un sc.nextLine() buit per consumir-lo.

int edat = sc.nextInt();
sc.nextLine();  // ← consumeix el \n pendent
String nom = sc.nextLine();  // ← ara llegeix correctament

2. Sortida formatada: printf i String.format

Java ofereix dos mecanismes per formatar la sortida de dades: System.out.printf() per imprimir directament i String.format() per construir una cadena formatada que podem guardar o enviar on vulguem. Tots dos usen els mateixos especificadors de format.

2.1 Especificadors de format principals
EspecificadorTipusExempleResultat
%dEnter (int, long)printf("%d", 42)42
%fDecimal (float, double)printf("%f", 3.14)3,140000
%.2fDecimal amb 2 decimalsprintf("%.2f", 3.14159)3,14
%sCadena (String)printf("%s", "hola")hola
%cCaràcter (char)printf("%c", 'A')A
%bBooleàprintf("%b", true)true
%nSalt de línia (portable)printf("fi%n")(nova línia)
%10dEnter ample 10, alineat dretaprintf("%10d", 42)        42
%-10sText ample 10, alineat esquerraprintf("%-10s|", "hola")hola      |
2.2 Exemples de printf i String.format
// printf: imprimeix directament System.out.printf("Nom: %-15s Edat: %3d Nota: %.2f%n", "Anna", 22, 8.75); System.out.printf("Nom: %-15s Edat: %3d Nota: %.2f%n", "Bernat", 19, 6.50); System.out.printf("Nom: %-15s Edat: %3d Nota: %.2f%n", "Clàudia", 21, 9.00); // Resultat: // Nom: Anna Edat: 22 Nota: 8,75 // Nom: Bernat Edat: 19 Nota: 6,50 // Nom: Clàudia Edat: 21 Nota: 9,00 // String.format: construeix una cadena String linia = String.format("[%05d] %-20s → %.2f €", 1, "Ordinador portàtil", 899.99); System.out.println(linia); // [00001] Ordinador portàtil → 899,99 €
Quan usar cada un? Usa printf quan vols imprimir immediatament. Usa String.format quan necessites la cadena per guardar-la, enviar-la a un fitxer o construir un missatge més complex.

3. La classe File

La classe File del paquet java.io representa un fitxer o directori del sistema de fitxers. No serveix per llegir ni escriure contingut, sinó per inspeccionar i manipular l'estructura: saber si existeix, quin nom té, crear directoris, llistar fitxers, esborrar...

Concepte clau: un objecte File és simplement un camí (path). Pot apuntar a un fitxer que no existeixi encara. Crear l'objecte no crea el fitxer; cal cridar createNewFile() o mkdir() explícitament.
3.1 Mètodes principals de File
MètodeDescripció
exists()Retorna true si el fitxer/directori existeix
isFile()Retorna true si és un fitxer (no directori)
isDirectory()Retorna true si és un directori
getName()Retorna el nom del fitxer o directori (sense ruta)
getAbsolutePath()Retorna la ruta absoluta completa
length()Retorna la mida en bytes
createNewFile()Crea un fitxer buit; retorna true si s'ha creat
mkdir()Crea el directori; retorna true si s'ha creat
mkdirs()Crea el directori i tots els intermedis necessaris
delete()Esborra el fitxer o directori (si és buit)
listFiles()Retorna un array File[] amb el contingut del directori
renameTo(File dest)Reanomena o mou el fitxer
3.2 Crear i inspeccionar fitxers
import java.io.File; import java.io.IOException; public class ExempleFile { public static void main(String[] args) throws IOException { File fitxer = new File("dades.txt"); if (!fitxer.exists()) { boolean creat = fitxer.createNewFile(); System.out.println("Fitxer creat: " + creat); } System.out.println("Nom: " + fitxer.getName()); System.out.println("Ruta abs.: " + fitxer.getAbsolutePath()); System.out.println("Mida (bytes):" + fitxer.length()); System.out.println("És fitxer: " + fitxer.isFile()); } }
3.3 Crear directoris i llistar contingut
import java.io.File; public class GestioDirectoris { public static void main(String[] args) { // Crear un directori nou File carpeta = new File("projectes/java"); if (carpeta.mkdirs()) { System.out.println("Directori creat: " + carpeta.getAbsolutePath()); } // Llistar el contingut del directori actual File actual = new File("."); File[] contingut = actual.listFiles(); if (contingut != null) { System.out.println("\nContingut del directori actual:"); for (File f : contingut) { String tipus = f.isDirectory() ? "[DIR] " : "[FIT] "; System.out.printf("%s %-30s %8d bytes%n", tipus, f.getName(), f.length()); } } } }
3.4 Llistar fitxers recursivament
// Mètode recursiu que llista tots els fitxers d'una carpeta i les seves subcarpetes public static void llistarRecursiu(File carpeta, int nivell) { File[] elements = carpeta.listFiles(); if (elements == null) return; for (File f : elements) { String sagna = " ".repeat(nivell); if (f.isDirectory()) { System.out.println(sagna + "📁 " + f.getName()); llistarRecursiu(f, nivell + 1); // crida recursiva } else { System.out.printf("%s📄 %-25s (%d bytes)%n", sagna, f.getName(), f.length()); } } }

4. Exercicis

E1 Calculadora de nòmina Bàsic

Escriu un programa que demani a l'usuari el nom d'un treballador, el seu sou brut mensual i el percentatge d'IRPF. El programa ha de mostrar una nòmina formatada amb printf.

  1. Llegeix nom (String), sou brut (double) i IRPF % (double) amb Scanner.
  2. Calcula la retenció i el sou net.
  3. Mostra la nòmina amb columnes alineades usant printf i %.2f.
Bàsic CA 5.1 · 5.2
✅ Solució — Calculadora de nòmina
import java.util.Scanner; public class Nomina { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Nom del treballador: "); String nom = sc.nextLine(); System.out.print("Sou brut mensual (€): "); double brut = sc.nextDouble(); System.out.print("Percentatge IRPF (%): "); double irpf = sc.nextDouble(); sc.close(); double retencio = brut * irpf / 100; double net = brut - retencio; System.out.println("\n========== NÒMINA =========="); System.out.printf("Treballador: %-20s%n", nom); System.out.printf("Sou brut: %10.2f €%n", brut); System.out.printf("Retenció IRPF(%4.1f%%): %6.2f €%n", irpf, retencio); System.out.println("----------------------------"); System.out.printf("SOU NET: %10.2f €%n", net); System.out.println("============================"); } }
E2 Taula de multiplicar formatada Bàsic

Demana a l'usuari un número enter entre 1 i 10 i mostra la taula de multiplicar d'aquell número amb la sortida perfectament alineada usant printf.

  1. Valida que el número estigui entre 1 i 10 (mostra un error si no).
  2. Mostra la taula del 1 al 10 amb format: 7 x 3 = 21
  3. Tots els números han d'ocupar el mateix ample de columna.
Bàsic CA 5.1 · 5.2
✅ Solució — Taula de multiplicar formatada
import java.util.Scanner; public class TaulaMultiplicar { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Introdueix un número (1-10): "); int n = sc.nextInt(); sc.close(); if (n < 1 || n > 10) { System.out.println("Error: el número ha d'estar entre 1 i 10."); return; } System.out.printf("%n=== Taula del %d ===%n", n); for (int i = 1; i <= 10; i++) { System.out.printf("%2d x %2d = %3d%n", n, i, n * i); } } }
E3 Inspector de directori Mitjà

Escriu un programa que demani a l'usuari una ruta de directori i mostri el seu contingut diferenciant fitxers de subcarpetes, indicant la mida de cada fitxer.

  1. Comprova que la ruta existeix i és un directori; si no, mostra un error.
  2. Llista el contingut amb listFiles().
  3. Mostra primer els directoris (prefix [DIR]) i després els fitxers ([FIT]), amb la mida en bytes formatada amb %10d.
  4. Mostra al final el total de fitxers i la mida total.
Mitjà CA 5.1 · 5.3
✅ Solució — Inspector de directori
import java.io.File; import java.util.Scanner; public class InspectorDirectori { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Introdueix la ruta del directori: "); String ruta = sc.nextLine(); sc.close(); File dir = new File(ruta); if (!dir.exists() || !dir.isDirectory()) { System.out.println("Error: la ruta no existeix o no és un directori."); return; } File[] elements = dir.listFiles(); if (elements == null || elements.length == 0) { System.out.println("El directori és buit."); return; } System.out.printf("%n%-40s %10s%n", "Nom", "Mida"); System.out.println("-".repeat(52)); long totalMida = 0; int totalFitxers = 0; // Primer directoris for (File f : elements) { if (f.isDirectory()) System.out.printf("[DIR] %-35s %10s%n", f.getName(), "-"); } // Després fitxers for (File f : elements) { if (f.isFile()) { System.out.printf("[FIT] %-35s %10d B%n", f.getName(), f.length()); totalMida += f.length(); totalFitxers++; } } System.out.println("-".repeat(52)); System.out.printf("Total: %d fitxer(s) — %d bytes%n", totalFitxers, totalMida); } }
E4 Organitzador de fitxers per extensió Avançat

Escriu un programa que llegeixi tots els fitxers d'un directori i els organitzi creant subdirectoris per extensió. Per exemple, tots els .txt aniran a la carpeta txt/, tots els .pdf a pdf/, etc.

  1. Llegeix la ruta del directori d'origen amb Scanner.
  2. Per a cada fitxer, obté la seva extensió (usa lastIndexOf('.')).
  3. Crea el subdirectori de l'extensió si no existeix (mkdir()).
  4. Mou el fitxer al subdirectori corresponent amb renameTo().
  5. Mostra un resum de quants fitxers s'han mogut per cada extensió.
Avançat CA 5.1 · 5.3
✅ Solució — Organitzador de fitxers per extensió
import java.io.File; import java.util.*; public class Organitzador { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Directori a organitzar: "); String ruta = sc.nextLine(); sc.close(); File dir = new File(ruta); if (!dir.isDirectory()) { System.out.println("No és un directori vàlid."); return; } Map<String, Integer> resum = new TreeMap<>(); for (File f : dir.listFiles()) { if (!f.isFile()) continue; String nom = f.getName(); int punt = nom.lastIndexOf('.'); String ext = (punt >= 0) ? nom.substring(punt + 1).toLowerCase() : "sense_ext"; File subdir = new File(dir, ext); if (!subdir.exists()) subdir.mkdir(); File desti = new File(subdir, nom); if (f.renameTo(desti)) { resum.merge(ext, 1, Integer::sum); } } System.out.println("\n=== Resum de l'organització ==="); resum.forEach((ext, count) -> System.out.printf(" .%-10s → %d fitxer(s)%n", ext, count)); } }
