1. Resum del bloc Entrada/Sortida

Abans de la pràctica integradora, fem un repàs ràpid de totes les classes i conceptes treballats en les activitats A1, A2 i A3. Cada grup cobreix un cas d'ús diferent.

Entrada consola

  • Scanner(System.in)
  • nextInt(), nextLine()
  • Problema \n pendent

Sortida formatada

  • printf(format, args)
  • String.format()
  • %d %f %s %.2f %-10s

Classe File

  • exists(), isFile()
  • mkdir(), mkdirs()
  • listFiles(), delete()

Fitxers text (escriptura)

  • FileWriter (bàsic)
  • PrintWriter (format)
  • BufferedWriter (eficient)

Fitxers text (lectura)

  • FileReader (caràcter)
  • Scanner(File) (tokens)
  • BufferedReader (línies)

Fitxers binaris

  • DataOutputStream
  • DataInputStream
  • EOFException

Serialització

  • implements Serializable
  • ObjectOutputStream
  • ObjectInputStream
Bones pràctiques: usa sempre try-with-resources (try (Recurs r = new...)) per tancar automàticament els streams. Gestiona IOException per a operacions de fitxers i EOFException per a fitxers binaris sense capçalera de mida.

2. Exercicis d'integració

E1 Registre de vendes en CSV Bàsic

Crea un programa que gestioni un registre de vendes en format CSV (vendes.csv). Cada venda té: data, producte i import. L'aplicació ha de permetre afegir vendes i mostrar el total recaptat.

  1. Llegeix les dades d'una nova venda per consola (data com a String, producte, import).
  2. Afegeix la venda al fitxer CSV en mode append: data;producte;import.
  3. Llegeix tot el fitxer i mostra les vendes amb printf alineat.
  4. Calcula i mostra el total recaptat.
Bàsic CA 5.1 · 5.2 · 5.4
Solució — Registre de vendes en CSV
import java.io.*; import java.util.Scanner; public class RegistreVendes { static final String FITXER = "vendes.csv"; public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Data (dd/mm/aaaa): "); String data = sc.nextLine(); System.out.print("Producte: "); String producte = sc.nextLine(); System.out.print("Import (euros): "); double import_ = sc.nextDouble(); sc.close(); try (PrintWriter pw = new PrintWriter(new FileWriter(FITXER, true))) { pw.printf("%s;%s;%.2f%n", data, producte, import_); } catch (IOException e) { System.out.println("Error: " + e.getMessage()); return; } double total = 0; System.out.printf("%n%-12s %-20s %10s%n", "Data", "Producte", "Import"); System.out.println("-".repeat(44)); try (BufferedReader br = new BufferedReader(new FileReader(FITXER))) { String linia; while ((linia = br.readLine()) != null) { String[] p = linia.split(";"); if (p.length == 3) { double imp = Double.parseDouble(p[2].replace(",", ".")); System.out.printf("%-12s %-20s %10.2f euros%n", p[0], p[1], imp); total += imp; } } } catch (IOException e) { System.out.println("Error: " + e.getMessage()); } System.out.println("-".repeat(44)); System.out.printf("%-32s %10.2f euros%n", "TOTAL RECAPTAT:", total); } }
E2 Backup de directori Mitjà

Crea un programa que llegeixi tots els fitxers .txt d'un directori d'origen i en faci una còpia de backup en un directori de destí, afegint la data d'avui al nom del fitxer.

  1. Demana el directori d'origen i el de destí per consola. Crea el destí si no existeix.
  2. Filtra els fitxers .txt del directori d'origen amb listFiles().
  3. Per cada fitxer, llegeix el contingut sencer amb BufferedReader.
  4. Escriu una còpia al destí amb el nom original_AAAA-MM-DD.txt usant BufferedWriter.
  5. Mostra un resum de quants fitxers s'han copiat i la mida total.
Mitjà CA 5.3 · 5.4 · 5.5
Solució — Backup de directori
import java.io.*; import java.time.LocalDate; import java.util.Scanner; public class BackupDirectori { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Directori origen: "); File origen = new File(sc.nextLine()); System.out.print("Directori desti: "); File desti = new File(sc.nextLine()); sc.close(); if (!origen.isDirectory()) { System.out.println("El directori d'origen no existeix."); return; } if (!desti.exists()) desti.mkdirs(); String data = LocalDate.now().toString(); int copiats = 0; long midaTotal = 0; for (File f : origen.listFiles()) { if (!f.isFile() || !f.getName().endsWith(".txt")) continue; String nomBase = f.getName().replace(".txt", ""); File fitxerDesti = new File(desti, nomBase + "_" + data + ".txt"); try (BufferedReader br = new BufferedReader(new FileReader(f)); BufferedWriter bw = new BufferedWriter(new FileWriter(fitxerDesti))) { String linia; while ((linia = br.readLine()) != null) { bw.write(linia); bw.newLine(); } copiats++; midaTotal += f.length(); System.out.printf("Copiat: %-25s -> %s%n", f.getName(), fitxerDesti.getName()); } catch (IOException e) { System.out.printf("Error copiant %s: %s%n", f.getName(), e.getMessage()); } } System.out.printf("%nResum: %d fitxer(s) copiats, %d bytes totals%n", copiats, midaTotal); } }
E3 Gestor d'inventari amb doble persistència Avançat

Projecte integrador que combina tots els conceptes del bloc E/S. Crea un gestor d'inventari de productes que guarda les dades en dos formats: un fitxer de text llegible (inventari.csv) i un fitxer binari serialitzat (inventari.dat).

  1. Carrega inicial: intenta carregar inventari.dat; si no existeix, carrega inventari.csv; si tampoc existeix, comença buit.
  2. Menú: 1) Afegir producte, 2) Llistar ordenat per preu descendent, 3) Buscar per nom, 4) Eliminar, 5) Guardar i sortir.
  3. Guardar: escriu el fitxer .dat serialitzat i genera també el .csv llegible.
  4. Llistar: mostra amb printf alineat i calcula el valor total de l'estoc.
  5. Log: cada operació (afegir, eliminar) s'afegeix a log.txt amb la data i hora.

La classe Producte té: nom, categoria, preu i estoc. Ha d'implementar Serializable.

Avançat CA 5.1 · 5.2 · 5.3 · 5.4 · 5.5
Solució — Gestor d'inventari amb doble persistència
import java.io.*; import java.time.*; import java.time.format.*; import java.util.*; // ─── Classe Producte ────────────────────────────── class Producte implements Serializable { private static final long serialVersionUID = 1L; private String nom, categoria; private double preu; private int estoc; public Producte(String nom, String cat, double preu, int estoc) { this.nom = nom; this.categoria = cat; this.preu = preu; this.estoc = estoc; } public String getNom() { return nom; } public String getCategoria() { return categoria; } public double getPreu() { return preu; } public int getEstoc() { return estoc; } public String toCSV() { return String.format("%s;%s;%.2f;%d", nom, categoria, preu, estoc); } public static Producte fromCSV(String linia) { String[] p = linia.split(";"); return new Producte(p[0], p[1], Double.parseDouble(p[2].replace(",", ".")), Integer.parseInt(p[3])); } } // ─── Main ───────────────────────────────────────── public class GestorInventari { static final String DAT = "inventari.dat"; static final String CSV = "inventari.csv"; static final String LOG = "log.txt"; static final DateTimeFormatter FMT = DateTimeFormatter.ofPattern("dd/MM/yyyy HH:mm:ss"); public static void main(String[] args) { List<Producte> inv = carregar(); Scanner sc = new Scanner(System.in); int opcio; do { System.out.printf("%n=== Inventari (%d productes) ===%n", inv.size()); System.out.println("1.Afegir 2.Llistar 3.Buscar 4.Eliminar 5.Guardar i sortir"); System.out.print("Opcio: "); opcio = sc.nextInt(); sc.nextLine(); switch (opcio) { case 1 -> afegir(sc, inv); case 2 -> llistar(inv); case 3 -> buscar(sc, inv); case 4 -> eliminar(sc, inv); case 5 -> guardar(inv); } } while (opcio != 5); sc.close(); } static void llistar(List<Producte> inv) { if (inv.isEmpty()) { System.out.println("Inventari buit."); return; } System.out.printf("%n%-20s %-12s %9s %6s %12s%n", "Nom","Categoria","Preu","Estoc","Valor"); System.out.println("-".repeat(63)); inv.stream() .sorted(Comparator.comparingDouble(Producte::getPreu).reversed()) .forEach(p -> System.out.printf( "%-20s %-12s %9.2f %6d %12.2f%n", p.getNom(), p.getCategoria(), p.getPreu(), p.getEstoc(), p.getPreu() * p.getEstoc())); double total = inv.stream() .mapToDouble(p -> p.getPreu() * p.getEstoc()).sum(); System.out.println("-".repeat(63)); System.out.printf("%-42s %12.2f euros%n", "VALOR TOTAL:", total); } static void afegir(Scanner sc, List<Producte> inv) { System.out.print("Nom: "); String nom = sc.nextLine(); System.out.print("Categoria: "); String cat = sc.nextLine(); System.out.print("Preu: "); double preu = sc.nextDouble(); System.out.print("Estoc: "); int estoc = sc.nextInt(); sc.nextLine(); inv.add(new Producte(nom, cat, preu, estoc)); log("AFEGIT: " + nom); System.out.println("Producte afegit."); } static void buscar(Scanner sc, List<Producte> inv) { System.out.print("Nom a cercar: "); String cerca = sc.nextLine().toLowerCase(); inv.stream() .filter(p -> p.getNom().toLowerCase().contains(cerca)) .forEach(p -> System.out.printf( " %-20s %.2f euros (estoc: %d)%n", p.getNom(), p.getPreu(), p.getEstoc())); } static void eliminar(Scanner sc, List<Producte> inv) { System.out.print("Nom a eliminar: "); String nom = sc.nextLine(); boolean ok = inv.removeIf(p -> p.getNom().equalsIgnoreCase(nom)); if (ok) { log("ELIMINAT: " + nom); System.out.println("Eliminat."); } else System.out.println("No trobat."); } static void guardar(List<Producte> inv) { try (ObjectOutputStream oos = new ObjectOutputStream( new FileOutputStream(DAT))) { oos.writeObject(inv); } catch (IOException e) { System.out.println("Error .dat: " + e.getMessage()); } try (PrintWriter pw = new PrintWriter(new FileWriter(CSV))) { inv.forEach(p -> pw.println(p.toCSV())); } catch (IOException e) { System.out.println("Error .csv: " + e.getMessage()); } System.out.printf("Guardat: %d productes -> %s i %s%n", inv.size(), DAT, CSV); } @SuppressWarnings("unchecked") static List<Producte> carregar() { if (new File(DAT).exists()) { try (ObjectInputStream ois = new ObjectInputStream( new FileInputStream(DAT))) { List<Producte> l = (List<Producte>) ois.readObject(); System.out.printf("Carregat des de %s (%d productes).%n", DAT, l.size()); return l; } catch (Exception ignored) {} } if (new File(CSV).exists()) { List<Producte> l = new ArrayList<>(); try (BufferedReader br = new BufferedReader(new FileReader(CSV))) { String lin; while ((lin = br.readLine()) != null) l.add(Producte.fromCSV(lin)); System.out.printf("Carregat des de %s (%d productes).%n", CSV, l.size()); } catch (IOException ignored) {} return l; } System.out.println("Inventari nou. Comenca buit."); return new ArrayList<>(); } static void log(String msg) { try (PrintWriter pw = new PrintWriter(new FileWriter(LOG, true))) { pw.printf("[%s] %s%n", LocalDateTime.now().format(FMT), msg); } catch (IOException ignored) {} } }
