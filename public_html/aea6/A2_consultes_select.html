1. La base de dades Gimnas

En aquesta activitat i les següents treballarem amb la base de dades Gimnas, que representa la gestió d'un gimnàs amb clients, activitats i inscripcions. Importa el fitxer Gimnas.sql a MySQL abans de continuar.

Clients (dni VARCHAR(10) PK, nom VARCHAR(50), edat INT, correu VARCHAR(100)) Activitats (idActivitat INT PK AUTO_INCREMENT, nom VARCHAR(50), nivell ENUM('inicial','mitjà','avançat'), places INT) Inscripcions (idInscripcio INT PK AUTO_INCREMENT, dni VARCHAR(10) FK → Clients, idActivitat INT FK → Activitats)

2. Statement i executeQuery()

Per executar una consulta SELECT necessitem dos objectes: un Statement (que representa la sentència SQL) i un ResultSet (que conté les files retornades).

El mètode executeQuery() de Statement envia la consulta SQL al servidor i retorna un ResultSet amb totes les files resultants.

import java.sql.*; public class LlistarClients { public static void main(String[] args) { String url = "jdbc:mysql://localhost:3306/Gimnas"; String usuari = "root"; String pass = ""; String sql = "SELECT dni, nom, edat, correu FROM Clients"; try (Connection con = DriverManager.getConnection(url, usuari, pass); Statement st = con.createStatement(); ResultSet rs = st.executeQuery(sql)) { while (rs.next()) { String dni = rs.getString("dni"); String nom = rs.getString("nom"); int edat = rs.getInt("edat"); String correu = rs.getString("correu"); System.out.printf("%-12s %-20s %3d %s%n", dni, nom, edat, correu); } } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); } } }
📌 try-with-resources amb múltiples recursos: Podem obrir Connection, Statement i ResultSet dins del mateix try(...) separats per punt i coma. Es tancaran tots automàticament i en ordre invers (primer ResultSet, després Statement, finalment Connection).

3. Mètodes del ResultSet

El ResultSet és com una taula virtual. El cursor comença abans de la primera fila i cal cridar next() per avançar. Quan next() retorna false, hem arribat al final.

Mètode de desplaçamentDescripció
next()Avança una fila. Retorna false si no n'hi ha més.
previous()Retrocedeix una fila.
first() / last()Va a la primera / última fila.
absolute(n)Va directament a la fila n.
Tipus SQLMètode JavaExemple
VARCHAR, CHAR, ENUMgetString()rs.getString("nom")
INT, SMALLINTgetInt()rs.getInt("edat")
DECIMAL, FLOATgetDouble()rs.getDouble("nota")
BOOLEANgetBoolean()rs.getBoolean("actiu")
QualsevolgetString()Sempre funciona per a qualsevol tipus.
💡 Nom o índex? Podeu passar el nom de la columna (rs.getString("nom")) o la seva posició (rs.getString(2)). L'índex comença en 1, no en 0. Amb el nom el codi és més llegible; amb l'índex és lleugerament més eficient.

4. Exercicis

E1 Llistat d'activitats del gimnàs Bàsic

Escriu un programa que recuperi totes les activitats de la taula Activitats i les mostri per pantalla amb format de taula.

  1. Usa la classe Connexio del paquet persistencia per obtenir la connexió.
  2. Executa la consulta SELECT idActivitat, nom, nivell, places FROM Activitats.
  3. Recorre el ResultSet amb while(rs.next()).
  4. Mostra cada activitat amb format: ID | Nom | Nivell | Places. Usa printf per alinear les columnes.
Bàsic CA 9.4
Solució — Llistat d'activitats del gimnàs
package presentacio; import java.sql.*; import persistencia.Connexio; public class LlistarActivitats { public static void main(String[] args) { String sql = "SELECT idActivitat, nom, nivell, places FROM Activitats"; try (Statement st = Connexio.getConnexio().createStatement(); ResultSet rs = st.executeQuery(sql)) { System.out.printf("%-5s %-15s %-10s %s%n", "ID", "Nom", "Nivell", "Places"); System.out.println("-".repeat(40)); while (rs.next()) { System.out.printf("%-5d %-15s %-10s %d%n", rs.getInt("idActivitat"), rs.getString("nom"), rs.getString("nivell"), rs.getInt("places")); } } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); } } }
E2 Clients inscrits a una activitat Mitjà

Escriu un programa que mostri, per a cada activitat, la llista de clients inscrits. Has de fer una consulta amb JOIN entre les tres taules.

  1. Fes la consulta SQL que uneixi Inscripcions, Clients i Activitats amb els JOIN corresponents.
  2. Ordena el resultat per nom d'activitat (ORDER BY Activitats.nom).
  3. Mostra: nom de l'activitat, nom del client i edat, agrupant visualment per activitat.
  4. Compta el nombre de clients per activitat i mostra'l al final de cada grup.
Mitjà CA 9.4 · 9.6
Solució — Clients inscrits a una activitat
String sql = "SELECT a.nom AS activitat, c.nom AS client, c.edat " + "FROM Inscripcions i " + "JOIN Clients c ON i.dni = c.dni " + "JOIN Activitats a ON i.idActivitat = a.idActivitat " + "ORDER BY a.nom, c.nom"; try (Statement st = Connexio.getConnexio().createStatement(); ResultSet rs = st.executeQuery(sql)) { String activitatActual = ""; int comptador = 0; while (rs.next()) { String activitat = rs.getString("activitat"); String client = rs.getString("client"); int edat = rs.getInt("edat"); if (!activitat.equals(activitatActual)) { if (!activitatActual.isEmpty()) { System.out.println(" Total inscrits: " + comptador); } System.out.println("\n▸ " + activitat); System.out.println(" " + "-".repeat(30)); activitatActual = activitat; comptador = 0; } System.out.printf(" %-20s (%d anys)%n", client, edat); comptador++; } if (!activitatActual.isEmpty()) { System.out.println(" Total inscrits: " + comptador); } } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); }
E3 Estadístiques d'ocupació del gimnàs Avançat

Crea un programa que mostri un resum d'ocupació de cada activitat: places totals, places ocupades i places lliures, usant consultes amb COUNT i LEFT JOIN.

  1. Fes una consulta que compti els inscrits per activitat amb COUNT i GROUP BY.
  2. Usa LEFT JOIN per incloure activitats sense cap inscrit.
  3. Calcula les places lliures = places totals - inscrits. Mostra'n els resultats en una taula formatada.
  4. Afegeix una línia final amb els totals globals del gimnàs.
Avançat CA 9.4 · 9.6
Solució — Estadístiques d'ocupació del gimnàs
String sql = "SELECT a.nom, a.places, COUNT(i.idInscripcio) AS inscrits " + "FROM Activitats a " + "LEFT JOIN Inscripcions i ON a.idActivitat = i.idActivitat " + "GROUP BY a.idActivitat, a.nom, a.places " + "ORDER BY a.nom"; try (Statement st = Connexio.getConnexio().createStatement(); ResultSet rs = st.executeQuery(sql)) { System.out.printf("%-15s %7s %8s %7s%n", "Activitat", "Places", "Inscrits", "Lliures"); System.out.println("-".repeat(42)); int totalPlaces = 0, totalInscrits = 0; while (rs.next()) { String nom = rs.getString("nom"); int places = rs.getInt("places"); int inscrits = rs.getInt("inscrits"); int lliures = places - inscrits; System.out.printf("%-15s %7d %8d %7d%n", nom, places, inscrits, lliures); totalPlaces += places; totalInscrits += inscrits; } System.out.println("-".repeat(42)); System.out.printf("%-15s %7d %8d %7d%n", "TOTAL", totalPlaces, totalInscrits, totalPlaces - totalInscrits); } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); }
