INSERT, UPDATE, DELETE i PreparedStatement

Modificació de dades amb executeUpdate(). Sentències parametritzades amb PreparedStatement per inserir nous clients, actualitzar places i eliminar inscripcions al Gimnàs.

1. Sentències de modificació

Per a operacions que modifiquen dades (INSERT, UPDATE, DELETE) no usem executeQuery() sinó executeUpdate(). La diferència clau és que executeUpdate() no retorna un ResultSet, sinó un int amb el nombre de files afectades.

try (Connection con = Connexio.getConnexio(); Statement st = con.createStatement()) { int filesAfectades = st.executeUpdate( "UPDATE Activitats SET places = 15 WHERE idActivitat = 1"); System.out.println("Files actualitzades: " + filesAfectades); } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); }

2. PreparedStatement

La classe Statement té un problema important: si construïm la sentència SQL concatenant valors de l'usuari, el codi és vulnerable a SQL Injection i ineficient si s'executa moltes vegades. La solució és usar PreparedStatement.

Amb PreparedStatement la sentència es precompila una sola vegada amb ? com a marcadors de posició, i els valors s'assignen després amb mètodes setTipus().

// ✗ MAL: vulnerable a SQL Injection String sql = "INSERT INTO Clients VALUES ('" + dni + "', '" + nom + "', ...)"; // ✓ BÉ: amb PreparedStatement String sql = "INSERT INTO Clients (dni, nom, edat, correu) VALUES (?, ?, ?, ?)"; try (PreparedStatement ps = Connexio.getConnexio().prepareStatement(sql)) { ps.setString(1, "67890123G"); // ? posició 1 ps.setString(2, "Carles Font"); // ? posició 2 ps.setInt (3, 31); // ? posició 3 ps.setString(4, "carles@gm.com");// ? posició 4 int files = ps.executeUpdate(); System.out.println("Client inserit. Files: " + files); } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); }
Mètode setTipus SQLExemple
setString(pos, val)VARCHAR, CHAR, ENUMps.setString(1, "Anna")
setInt(pos, val)INT, SMALLINTps.setInt(2, 25)
setDouble(pos, val)DECIMAL, FLOATps.setDouble(3, 8.5)
setBoolean(pos, val)BOOLEANps.setBoolean(4, true)
⚠️ Els índexs dels paràmetres comencen en 1, no en 0. El primer ? és la posició 1, el segon la posició 2, etc.
📌 PreparedStatement per a SELECT: PreparedStatement no és només per a modificacions. Si la consulta SELECT té condicions que depenen de valors externs (per exemple, buscar un client per DNI), cal usar PreparedStatement amb executeQuery() per evitar SQL Injection.

3. Exemples complets

Inserir un nou client

String sql = "INSERT INTO Clients (dni, nom, edat, correu) VALUES (?, ?, ?, ?)"; try (PreparedStatement ps = Connexio.getConnexio().prepareStatement(sql)) { ps.setString(1, "67890123G"); ps.setString(2, "Carles Font"); ps.setInt (3, 31); ps.setString(4, "carles@gmail.com"); int files = ps.executeUpdate(); System.out.println("Inserit! Files afectades: " + files); } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); }

Actualitzar les places d'una activitat

String sql = "UPDATE Activitats SET places = ? WHERE idActivitat = ?"; try (PreparedStatement ps = Connexio.getConnexio().prepareStatement(sql)) { ps.setInt(1, 20); // noves places ps.setInt(2, 1); // idActivitat int files = ps.executeUpdate(); System.out.println("Actualitzat! Files: " + files); } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); }

Eliminar la inscripció d'un client

String sql = "DELETE FROM Inscripcions WHERE dni = ? AND idActivitat = ?"; try (PreparedStatement ps = Connexio.getConnexio().prepareStatement(sql)) { ps.setString(1, "12345678A"); ps.setInt (2, 3); int files = ps.executeUpdate(); if (files > 0) { System.out.println("Inscripció eliminada."); } else { System.out.println("No s'ha trobat la inscripció."); } } catch (SQLException e) { System.out.println("Error: " + e.getMessage()); }

4. Exercicis

E1 Afegir nous clients al gimnàs Bàsic

Escriu un programa que insereixi tres nous clients a la taula Clients del Gimnàs usant PreparedStatement, i comprovi que s'han inserit consultant el total de clients abans i després.

  1. Consulta i mostra el nombre de clients abans de la inserció.
  2. Crea un PreparedStatement per a l'INSERT i insereix els tres clients en un bucle.
  3. Consulta i mostra el nombre de clients després de la inserció.
  4. Verifica que la diferència és 3.
Bàsic CA 9.3
Solució — Afegir nous clients al gimnàs
import java.sql.*; import persistencia.Connexio; public class AfegirClients { public static void main(String[] args) throws SQLException { Connection con = Connexio.getConnexio(); // Recompte inicial try (Statement st = con.createStatement(); ResultSet rs = st.executeQuery("SELECT COUNT(*) FROM Clients")) { rs.next(); System.out.println("Clients abans: " + rs.getInt(1)); } // Inserció dels 3 nous clients String[][] clients = { {"67890123G", "Carles Font", "31", "carles@gm.com"}, {"78901234H", "Marta Puig", "26", "marta@gm.com" }, {"89012345I", "Andreu Sala", "38", "andreu@gm.com"} }; String sql = "INSERT INTO Clients (dni, nom, edat, correu) VALUES (?,?,?,?)"; try (PreparedStatement ps = con.prepareStatement(sql)) { for (String[] c : clients) { ps.setString(1, c[0]); ps.setString(2, c[1]); ps.setInt (3, Integer.parseInt(c[2])); ps.setString(4, c[3]); System.out.println("Inserit: " + c[1] + " (" + ps.executeUpdate() + " fila)"); } } // Recompte final try (Statement st = con.createStatement(); ResultSet rs = st.executeQuery("SELECT COUNT(*) FROM Clients")) { rs.next(); System.out.println("Clients després: " + rs.getInt(1)); } } }
E2 Gestió d'inscripcions Mitjà

Escriu un programa que permeti inscriure un client existent a una activitat i, posteriorment, cancel·lar-ne la inscripció. Ha de verificar que no hi ha places lliures abans d'inscriure.

  1. Consulta les places lliures de l'activitat amb id 2 (Crossfit): si no n'hi ha, mostra un avís i no inscriguis.
  2. Si hi ha places, inscriu el client 34567890C (Maria López) a l'activitat 2 amb un PreparedStatement.
  3. Confirma la inscripció mostrant el nombre de files afectades.
  4. Elimina la mateixa inscripció amb un segon PreparedStatement i mostra el resultat.
Mitjà CA 9.3 · 9.5
Solució — Gestió d'inscripcions
Connection con = Connexio.getConnexio(); int idActivitat = 2; String dni = "34567890C"; // 1. Comprovar places lliures String sqlPlaces = "SELECT a.places - COUNT(i.idInscripcio) AS lliures " + "FROM Activitats a LEFT JOIN Inscripcions i ON a.idActivitat = i.idActivitat " + "WHERE a.idActivitat = ? GROUP BY a.places"; try (PreparedStatement ps = con.prepareStatement(sqlPlaces)) { ps.setInt(1, idActivitat); try (ResultSet rs = ps.executeQuery()) { rs.next(); int lliures = rs.getInt("lliures"); System.out.println("Places lliures: " + lliures); if (lliures == 0) { System.out.println("L'activitat és plena. No es pot inscriure."); return; } } } // 2. Inscriure el client try (PreparedStatement ps = con.prepareStatement( "INSERT INTO Inscripcions (dni, idActivitat) VALUES (?, ?)")) { ps.setString(1, dni); ps.setInt (2, idActivitat); System.out.println("Inscripció creada. Files: " + ps.executeUpdate()); } // 3. Cancel·lar la inscripció try (PreparedStatement ps = con.prepareStatement( "DELETE FROM Inscripcions WHERE dni = ? AND idActivitat = ?")) { ps.setString(1, dni); ps.setInt (2, idActivitat); int files = ps.executeUpdate(); System.out.println(files > 0 ? "Inscripció cancel·lada." : "No s'ha trobat la inscripció."); }
E3 Actualització massiva de preus per nivell Avançat

El gimnàs decideix afegir una columna preu DECIMAL(5,2) a la taula Activitats i assignar preus automàticament segons el nivell. Desenvolupa el programa complet.

  1. Afegeix la columna preu a la taula Activitats executant un ALTER TABLE amb executeUpdate().
  2. Usa un PreparedStatement per actualitzar el preu de totes les activitats per nivell: inicial → 25.00 €, mitjà → 35.00 €, avançat → 45.00 €.
  3. Mostra el resultat final amb un SELECT de totes les activitats i el seu preu.
Avançat CA 9.3 · 9.5
Solució — Actualització massiva de preus per nivell
Connection con = Connexio.getConnexio(); // 1. Afegir columna preu try (Statement st = con.createStatement()) { st.executeUpdate("ALTER TABLE Activitats ADD COLUMN preu DECIMAL(5,2)"); System.out.println("Columna 'preu' afegida."); } catch (SQLException e) { // Si ja existeix la columna, ignorem l'error System.out.println("Nota: " + e.getMessage()); } // 2. Actualitzar preus per nivell String[][] nivells = { {"inicial", "25.00"}, {"mitjà", "35.00"}, {"avançat", "45.00"} }; try (PreparedStatement ps = con.prepareStatement( "UPDATE Activitats SET preu = ? WHERE nivell = ?")) { for (String[] n : nivells) { ps.setDouble(1, Double.parseDouble(n[1])); ps.setString(2, n[0]); int files = ps.executeUpdate(); System.out.printf("Nivell %-8s → %.2f € (%d files)%n", n[0], Double.parseDouble(n[1]), files); } } // 3. Mostrar resultat final try (Statement st = con.createStatement(); ResultSet rs = st.executeQuery( "SELECT nom, nivell, places, preu FROM Activitats ORDER BY nivell")) { System.out.println("\n--- Activitats amb preu ---"); while (rs.next()) { System.out.printf("%-15s %-8s %2d places %.2f €%n", rs.getString("nom"), rs.getString("nivell"), rs.getInt("places"), rs.getDouble("preu")); } }
