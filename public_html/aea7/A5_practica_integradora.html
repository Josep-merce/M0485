Implementació d'un sistema de gestió d'un institut DAW amb les entitats Alumne, Modul, Ra i Nota: model complet, inserció massiva, consultes JPQL i càlcul de nota final.

★ Pràctica avaluable Pt1 — AEA7

Aquesta activitat és la pràctica avaluable (Pt1) de l'AEA7. Es lliura un projecte NetBeans complet amb totes les entitats, el persistence.xml, la inserció de dades i les consultes implementades. El professor avaluarà el projecte comprimit lliurat al Classroom.

1. Enunciat

Volem una aplicació per gestionar les classes del cicle DAW i les notes dels alumnes. Necessitem la informació següent:

EntitatAtributsNotes
AlumnedniA (String, PK), nom, edatLa clau primària és el DNI (String), no autogenerat
ModulnumeroM (int, PK), nom, duradaLa clau primària és el número de mòdul (int)
Raid (Long, auto), numeroRA, nom, durada, pesPertany a un Modul (ManyToOne)
Notaid (Long, auto), nota (double)Pertany a un Alumne i a un Ra (ManyToOne × 2)
Relacions entre entitats
  • Modul → Ra: un mòdul té molts RAs (@OneToMany)
  • Alumne → Nota: un alumne té moltes notes (@OneToMany)
  • Ra → Nota: un RA té moltes notes (@OneToMany)
📌 Nota final: es calcula, no s'emmagatzema: La nota final del mòdul no s'ha de desar a la BD. Es calcula en temps d'execució navegant per les relacions: per a cada nota de l'alumne que pertanyi a un RA del mòdul indicat, es multiplica la nota pel pes del RA i es suma. Fórmula: notaFinal = SUM(nota_RA × pes_RA / 100).

2. Requisits de la pràctica

Part A — Model de dades (25%)
  1. Crea les 4 classes d'entitat (Alumne, Modul, Ra, Nota) amb les anotacions JPA correctes.
  2. Configura el fitxer persistence.xml amb totes les classes incloses a la unitat "InstitutPU" i la URL $objectdb/db/institut.odb.
Part B — Inserció massiva (25%)
  1. Crea 5 mòduls. Cada mòdul ha de tenir 3 RAs amb pesos que sumin 100 (per exemple: 33.33 + 33.33 + 33.34).
  2. Crea 10 alumnes. Assigna a cada alumne notes per a 5 RAs aleatoris (valors entre 4.0 i 10.0). Usa flush() + clear() adequadament.
Part C — Consultes JPQL (30%)
  1. Llista tots els alumnes ordenats per nom.
  2. Mostra tots els RAs del Mòdul 1 ordenats per número de RA.
  3. Llista les notes d'un alumne concret (per DNI) amb el nom del RA corresponent.
  4. Mostra la mitjana de les notes de cada RA (nom del RA i mitjana). Ordena per RA descendent.
Part D — Actualització i eliminació (20%)
  1. Incrementa en 1 punt (màxim 10) la nota de tots els alumnes en el RA amb pes més alt del Mòdul 1.
  2. Implementa el mètode calcularNotaFinalModul(Alumne, Modul) usant Streams i mostra la nota final del Mòdul 1 per a tots els alumnes.

⚠️ Elimina la BD entre execucions: Alumne i Modul usen claus primàries manuals (DNI i numeroM). Si executes InsercioMassiva un segon cop sense eliminar primer la BD, JPA llançarà una excepció per duplicat de clau primària. Obre el Database Explorer, elimina el fitxer institut.odb i torna a executar.
Inserció massiva — InsercioMassiva.java
public class InsercioMassiva { public static void main(String[] args) { EntityManagerFactory emf = Persistence.createEntityManagerFactory("InstitutPU"); EntityManager em = emf.createEntityManager(); em.getTransaction().begin(); Random rand = new Random(); List<Ra> rasTotals = new ArrayList<>(); int comptador = 0; // PRIMER: crear mòduls i RAs for (int m = 1; m <= 5; m++) { Modul modul = new Modul(m, "Mòdul " + m, 180); List<Ra> raList = new ArrayList<>(); for (int r = 1; r <= 3; r++) { double pes = (r < 3) ? 33.33 : 33.34; Ra ra = new Ra(r, "RA "+r+" del Mòdul "+m, 60, pes, modul); em.persist(ra); raList.add(ra); rasTotals.add(ra); comptador++; } modul.setRas(raList); em.persist(modul); comptador++; } // SEGON: crear alumnes amb notes for (int i = 1; i <= 10; i++) { String dni = String.format("%08dA", i); Alumne alumne = new Alumne(dni, "Alumne "+i, 18+rand.nextInt(5)); List<Nota> notes = new ArrayList<>(); Collections.shuffle(rasTotals); for (int j = 0; j < 5; j++) { Ra ra = rasTotals.get(j); double valor = 4.0 + rand.nextDouble() * 6.0; valor = Math.round(valor * 10.0) / 10.0; Nota n = new Nota(valor, alumne, ra); em.persist(n); notes.add(n); comptador++; } alumne.setNotes(notes); em.persist(alumne); comptador++; // flush + clear cada 10 objectes per alliberar memòria if (comptador % 10 == 0) { em.flush(); em.clear(); } } em.getTransaction().commit(); em.close(); emf.close(); System.out.println("Inserció completada: " + comptador + " objectes."); } }
Càlcul de la nota final de mòdul (Streams)
public static double calcularNotaFinalModul(Alumne alumne, Modul modul) { return alumne.getNotes().stream() .filter(n -> n.getRa().getModul().getNumeroM() == modul.getNumeroM()) .mapToDouble(n -> n.getNota() * n.getRa().getPes() / 100.0) .sum(); }
