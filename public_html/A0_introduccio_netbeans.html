1. Què és Java?

Java és un llenguatge de programació d'alt nivell, compilat i d'ús general creat el 1995 per James Gosling. El seu lema és "Escriu una vegada, executa a qualsevol lloc" (WORA): el codi compilat s'executa a qualsevol sistema que tingui una JVM instal·lada.

CaracterísticaDescripció
PortabilitatEl bytecode s'executa a Windows, macOS i Linux sense canvis.
Orientat a objectesTot s'organitza en classes i objectes.
SeguretatDissenyat per evitar vulnerabilitats comunes. Gestió automàtica de memòria (Garbage Collection).
Àmplia comunitatUn dels llenguatges més demandats al mercat laboral. Frameworks com Spring, ecosistema Maven/Gradle.
Versió del cursJDK 21 (LTS) — versió de suport a llarg termini, estable i recomanada.

2. JDK, JRE i JVM

ComponentNom completPer a què serveix
JDK Java Development Kit Conjunt complet d'eines per desenvolupar en Java. Inclou el compilador (javac), el JRE i les eines de depuració.
JRE Java Runtime Environment Entorn per executar programes Java (sense poder-los compilar). Inclou la JVM i les biblioteques estàndard.
JVM Java Virtual Machine Màquina virtual que interpreta el bytecode i el tradueix a instruccions del sistema operatiu.
Procés de compilació i execució:
Codi font (.java) → javac → Bytecode (.class) → JVM → Execució al sistema operatiu
💡 Quin JDK instal·lar? Sempre el JDK 21 (LTS). Les versions LTS (Long-Term Support) reben actualitzacions de seguretat durant anys i són les recomanades per a projectes seriosos i entorns educatius.

3. Instal·lació del JDK i NetBeans

1 Descarregar el JDK 21 Ves a oracle.com/java/technologies/downloads/#java21 i descarrega l'instal·lador per al teu sistema operatiu (Windows .exe, macOS .dmg, Linux .tar.gz).
Pas 1
2 Instal·lar el JDK Executa l'instal·lador i segueix l'assistent. Deixa la ruta per defecte. Verifica: obre una terminal i executa java -version.
Pas 2
3 Descarregar Apache NetBeans Ves a netbeans.apache.org, descarrega la versió més recent i instal·la-la. NetBeans detectarà automàticament el JDK instal·lat.
Pas 3
4 Configuració opcional: tema fosc Tools → Options → Appearance → Look and Feel → FlatLaf Dark. Millora la visibilitat i redueix la fatiga visual en sessions llargues.
Pas 4

4. Primer projecte: Hola Món

Segueix aquests passos a NetBeans per crear el primer projecte Java i executar el programa Hola Món.

1 Crear el projecte File → New Project → Java with Ant → Java Application. Nom: IntroduccioJava. Desmarca "Create Main Class".
NetBeans
2 Crear la classe A Source Packages → clic dret → New → Java Class. Nom: HolaMundo. NetBeans crea el fitxer HolaMundo.java.
NetBeans
3 Escriure el codi Escriu el codi de l'exemple de sota. Observa l'estructura: classe, mètode main i instruccions dins el cos.
Java
4 Executar Clic dret sobre HolaMundo.java → Run File. A la consola de sortida veuràs: Hola Món amb Java.
NetBeans
public class HolaMundo { public static void main(String[] args) { // Mostra un missatge per pantalla System.out.println("Hola Món amb Java"); System.out.println("Benvinguts al curs!"); } }
Estructura bàsica d'un programa Java:
• public class HolaMundo → defineix la classe. El nom del fitxer ha de coincidir exactament (HolaMundo.java).
• public static void main(String[] args) → punt d'entrada del programa. Java sempre comença l'execució per aquí.
• System.out.println(...) → mostra text per pantalla i fa un salt de línia.
• Cada instrucció acaba en punt i coma ;.

5. Conceptes clau de l'entorn de desenvolupament

ConcepteDescripció
IDEEntorn de Desenvolupament Integrat. Combina editor de codi, compilador, depurador i gestió de projectes en una sola aplicació.
Editor de codiRessaltat de sintaxi, autocompletat i detecció d'errors en temps real.
Compilador integratNetBeans compila automàticament en guardar (Compile on Save). Errors de compilació apareixen subratllats en vermell.
DepuradorPermet executar el programa pas a pas, establir punts d'interrupció i inspeccionar el valor de les variables.
Consola de sortidaFinestra inferior on es mostra el resultat de System.out.println() i els errors d'execució.
💡 Bones pràctiques des del primer dia: activa Compile on Save (clic dret al projecte → Properties → Build → Compile) i augmenta la mida de la font a 15 (Tools → Options → Fonts & Colors) per llegir millor el codi.
